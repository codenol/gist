# Инструкции по работе с семантическими токенами

## Для кого это?

Эти инструкции созданы для дизайнеров, которые работают с дизайн-токенами и создают компонентные токены на основе семантических.

> ⚠️ **Важно:** Если ты впервые работаешь с токенами, начни с **[Инструкции по компонентному подходу к токенам](../component_approach_guide.md)** — она показывает, как именно мы используем токены на практике с примерами для кнопки, инпута, таблицы и бейджа.

## Какие подходы есть?

У нас есть 5 подходов к организации семантических токенов:

1. **[По функциональности](functional.md)** — токены организованы по функциональному назначению (background, text, border, action, status)
2. **[По ролям](roles.md)** — токены организованы по ролям в системе (brand, neutral, status, interactive, surface)
3. **[По иерархии](hierarchy.md)** — токены организованы по уровням важности (level_1, level_2, level_3)
4. **[По контексту](context.md)** — семантический слой не используется, компонентные токены ссылаются напрямую на базовый слой
5. **[Простой подход](simple.md)** — максимально упрощенная версия без категорий (primary, secondary, text, border)

## Как выбрать подход?

Выбор зависит от твоих потребностей:

- **Нужна интуитивность?** → [Функциональность](functional.md)
- **Нужен White Label?** → [Роли](roles.md)
- **Важна иерархия?** → [Иерархия](hierarchy.md)
- **Нужен быстрый старт?** → [Контекст](context.md) или [Простой](simple.md)
- **Нужна максимальная простота?** → [Простой](simple.md)

Подробнее о сравнении подходов см. [главный README](../README.md).

## Структура инструкций

Каждая инструкция содержит:

1. **Что это за подход?** — краткое описание
2. **Как это работает?** — основная идея и структура
3. **Как выбирать токены?** — пошаговая инструкция
4. **Примеры использования** — конкретные примеры
5. **Spacing, Typography, Border Radius** — как работать с другими свойствами
6. **Преимущества и недостатки** — что хорошо, что плохо
7. **Когда использовать?** — рекомендации
8. **Примеры компонентов** — ссылки на конкретные примеры
