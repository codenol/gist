1. **Исправление структуры коллекций** (`01-fix-collections.md`)
   - Убедиться, что коллекция "Color" имеет режимы Light и Dark
   - Распределить токены по режимам

2. **Создание новых токенов** (по приоритету)

   - Badge и Status: `03-create-badge-status-tokens.md`
   - Остальные: `04-create-other-tokens.md`

3. **Миграция компонентов** (по приоритету)
   - Button: `05-migrate-button.md`
   - Badge и Status: `06-migrate-badge-status.md`
   - Остальные: `07-migrate-other-components.md`

4. **Финальная проверка** (`08-final-check.md`)
   - Проверка всех компонентов
   - Проверка макетов
   - Обновление документации
