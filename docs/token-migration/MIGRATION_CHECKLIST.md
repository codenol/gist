# Чеклист миграции токенов

Используйте этот чеклист для отслеживания прогресса миграции.

## Подготовка

- [ ] Токены экспортированы из Figma
- [ ] Инвентарь токенов создан (`token-inventory.md`)
- [ ] Файл маппинга создан (`migration-mapping.json`)
- [ ] Приоритеты миграции определены

## Исправление структуры коллекций

- [ ] Коллекция "Color" имеет режимы Light и Dark
- [ ] Все токены правильно распределены по режимам
- [ ] Другие коллекции проверены
